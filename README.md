Nutrizen – .NET MAUI

Aplicación móvil desarrollada con .NET MAUI orientada a la gestión y pedido de comida saludable, permitiendo a los usuarios explorar un catálogo de alimentos, personalizar pedidos y realizar compras de manera sencilla desde dispositivos Android e iOS.

Descripción del proyecto

Nutrizen  es una aplicación multiplataforma diseñada para fomentar hábitos alimenticios saludables mediante una experiencia intuitiva de compra de alimentos balanceados. El sistema simula un flujo completo de tienda de comida saludable, desde la exploración del menú hasta la confirmación del pedido.

Objetivo

Desarrollar una aplicación móvil utilizando .NET MAUI que permita:

Mostrar un catálogo de alimentos saludables
Gestionar productos y categorías
Simular un sistema de pedidos
Proporcionar una interfaz moderna, limpia y responsiva
Aplicar principios de programación orientada a objetos (POO)

Tecnologías utilizadas
.NET MAUI (.NET 7/8)
C#
XAML
HTML
CSS
SQL SERVER
Visual Studio 2022+

Funcionalidades
-Catálogo de alimentos saludables
-Búsqueda y filtrado de productos
-Carrito de compras
-Detalle de producto
-Simulación de pago
-Perfil de usuario 
-Historial de pedidos
